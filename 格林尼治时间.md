### Greenwich time
struct tm中记录着时间的所有标准信息。tm_year,tm_mon,tm_day……
+ time(time_t*)获得当前秒数
+ gtime(time_t*)得到结构体
+ mktime(struct tm)得到秒数
### 使用用例
```c++
#include<fun.c>

int main(){
  time_t now;
  time(&now);
  struct tm=gtime(&now);
  printf("%04d %02d %02d %02d:%02d:%02d %d %ld",p->tm_year+1900,p->tm_mon+1,p->tm_mday,p->tm_hour,p->tm_min,p->tm_sec,p->tm_wday,mktime(p)); 
}
```
